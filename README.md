# GameHub100 - 100个小游戏合集

🎮 一个包含100个精选小游戏的网页合集，完全免费开源！

## 🌟 特点

- ✅ **100个精选游戏** - 涵盖10大类别
- ✅ **完全免费** - 无需注册，打开即玩
- ✅ **响应式设计** - 支持PC和移动端
- ✅ **开源项目** - 代码完全开放
- ✅ **无需后端** - 纯前端实现

## 🎮 游戏分类

1. **🧩 益智类** (10个) - 锻炼大脑思维
2. **🎯 动作类** (10个) - 考验反应速度
3. **♟️ 策略类** (10个) - 规划战术策略
4. **⚡ 反应类** (10个) - 挑战反应极限
5. **🎯 射击类** (10个) - 瞄准射击乐趣
6. **🎲 休闲类** (10个) - 轻松娱乐放松
7. **📝 文字类** (10个) - 文字游戏挑战
8. **🃏 卡牌类** (10个) - 经典纸牌游戏
9. **⚽ 体育类** (10个) - 运动竞技体验
10. **🎨 创意类** (10个) - 发挥创意想象

## 🛠️ 技术栈

- **HTML5** - 语义化标签
- **CSS3** - Flexbox/Grid布局
- **JavaScript** - ES6+原生实现
- **Canvas** - 游戏渲染
